# Contributing

Guppi's development happens in a private repository; this public repo is the
distribution channel — the installer, versioned releases, and the issue
tracker. **Pull requests here can't be merged** (there is no source tree to
merge into), so please don't invest time in one.

What genuinely helps:

- **Bug reports** with the release version and logs — install failures on new
  hardware are especially valuable while the installer is young.
- **Instrument support requests** with an `*IDN?` string and a programming
  manual link. Drivers are prioritized largely by these.
- **Ideas and questions** in Discussions.

Source in the release tarballs is provided for transparency and reference —
see [LICENSE](LICENSE) for what that does and doesn't permit.
